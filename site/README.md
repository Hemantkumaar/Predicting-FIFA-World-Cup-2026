This directory contains a simple static copy of the simulation pages.

## Accessing the site

Open `index.html` in a web browser or start a local server:

```sh
cd site
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.

The cover page displays a title and links to the individual stage pages.

This folder is intended for quick testing and preview; the source HTML
files under `Simulation/` in the project root are the originals.
# Predicting-FIFA-World-Cup-2026
Predicting FIFA World Cup 2026 match outcomes using historical football data, SQL database integration, and machine learning in Python. Combines team performance, rankings, player statistics, and weather conditions to model match probabilities and analyze how contextual factors influence tournament results.
